# DeepPly Frontend

This is the user interface for DeepPly, a chess analysis assistant powered by AI. Built using Flutter, it allows users to upload PGN files, view annotated results, and chat with an AI about their mistakes.

### Features:
- PGN uploader and board viewer
- Game summary + mistake insights
- Interactive chat with AI coach
- Clean, responsive UI
